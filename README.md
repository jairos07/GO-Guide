# Go Language Learning Guide

This repository contains a detailed introductory guide to the Go programming language.  
It is designed for students and beginners who want to understand Go’s syntax, structure, and fundamental programming concepts.

---

## Overview

The document `Guide.md` provides a complete explanation of Go basics, including practical code examples and line-by-line analysis.  
It introduces key topics such as variable declarations, constants, comments, and output functions.

---

## Contents

### 1. Introduction
- Structure of a Go program  
- The `main` package and the `fmt` import  
- Writing a basic “Hello World” example

### 2. Declarations and Formatting
- How Go handles semicolons and line breaks  
- Writing compact Go code  
- Correct declaration syntax

### 3. Comments
- Single-line (`//`) and multi-line (`/* ... */`) comments  
- Best practices for code readability

### 4. Variables
- Basic types: `int`, `float32`, `string`, `bool`  
- Variable declaration using `var` and `:=`  
- Assigning values and declaring multiple variables  
- Rules and conventions for variable names

### 5. Constants
- Declaring constants with `const`  
- Defined and undefined constants  
- Immutability and naming conventions

### 6. Output Functions
- Using `fmt.Print()`, `fmt.Println()`, and `fmt.Printf()`  
- Formatting and printing multiple values  
- Handling new lines and spacing

---

## Features
- Step-by-step explanations for each concept  
- Real and functional Go examples  
- Beginner-oriented structure and language
